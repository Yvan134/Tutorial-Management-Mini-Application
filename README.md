#  Tutorial Management Mini Application — Angular 13

A front‑end application built with **Angular 13** that allows users to create, view, update, search, and delete tutorials.  
This project demonstrates a complete CRUD workflow, the use of Angular services, REST API integration, and reactive data handling with **RxJS**.

---

##  Project Objectives
- Build a modular and maintainable Angular application  
- Implement a full CRUD system using Angular components and services  
- Consume a REST API using `HttpClient`  
- Manage data reactively with RxJS Observables  
- Apply Angular best practices for project structure and component communication  

---

## Tech Stack
- **Angular 13**  
- **TypeScript**  
- **RxJS**  
- **HTML / CSS**  
- **Bootstrap** (optional)  
- **REST API** (provided for the project)  
- **HttpClientModule**

---

##  Features
- Add a new tutorial  
- Display all tutorials  
- Search tutorials by title  
- Edit an existing tutorial  
- Delete a tutorial  
- Delete all tutorials  
- Reactive data updates using RxJS

---

## Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/<your-username>/<repo-name>.git
cd <repo-name>
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the application
```bash
ng serve
```

The app will be available at:  
 http://localhost:4200/
